# Himatch (ヒマッチ)

友達の空き時間を自動マッチング & 文脈付き提案する予定調整アプリ

## コンセプト

「いつ空いてる？」を聞くアプリではなく、**「この日、こう遊べるよ！」と教えてくれるアプリ**

## 主要機能

- **シフト/スケジュール共有** - 友達のシフト・予定を一覧表示
- **グループ機能** - グループを作成し友達同士で共有
- **文脈付き候補日提案** - 空き時間帯と長さに基づく自動提案
  - 「3/15 みんな夜空き → 飲み会にぴったり！」
  - 「3/22 全員終日空き → 日帰り旅行もOK！」
  - 「3/29 午後だけ空き → ランチ or カフェ向き」
- **天気予報連携** (Phase 2) - 候補日の天気を表示しアクティビティを提案
